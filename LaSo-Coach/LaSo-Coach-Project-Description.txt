LaSo-Coach: Comprehensive Fitness Coaching Platform
==================================================

PROJECT OVERVIEW
================

LaSo-Coach is a sophisticated, full-stack fitness coaching platform designed to provide personalized training, nutrition guidance, and community support. The platform serves as a complete ecosystem connecting fitness coaches with clients through an intuitive, feature-rich application that combines real-time communication, progress tracking, gamification, and comprehensive health management.

CORE CONCEPT & BUSINESS MODEL
=============================

The platform operates on a subscription-based model where users pay for access to personalized coaching services, nutrition plans, fitness challenges, and community features. Coaches can manage multiple clients, create customized programs, and track client progress through an advanced admin interface. The system includes gamification elements to increase user engagement and retention.

TECHNICAL ARCHITECTURE
======================

Development Environment:
- Frontend: React 18 + TypeScript + Vite + Tailwind CSS
- Backend: Node.js + Express + TypeScript + PostgreSQL + Prisma ORM
- Admin Panel: React 19 + Vue 3 Hybrid + TanStack Query + Pinia
- Real-time: Socket.IO for WebSocket connections
- Payments: Stripe + PayPal integration
- File Storage: AWS S3
- Email: Nodemailer with SMTP templates
- Authentication: JWT with refresh tokens

Deployment Architecture:
- Development: Netlify (Frontend) + Render (Backend) + Neon (Database)
- Staging: Netlify (Frontend) + Render (Backend) + Neon (Database)
- Production: Hostinger (Frontend) + Render (Backend) + Neon (Database)
